## 第一章  机器学习概述

### 1.1 AI → ML → DL

```
AI（人工智能）→ ML（机器学习）→ DL（深度学习）
```

- **机器学习**：数据驱动，自动发现规律，用规律做预测/决策
- 多领域交叉：概率论、统计学、逼近论、凸优化、算法复杂度

### 1.2 发展简史

| 阶段 | 时期 | 标志 |
|------|------|------|
| 推理期 | 1950s-70s | 图灵测试、感知机(罗森布拉特) |
| 知识驱动 | 1970s-80s | 专家系统(MYCIN)、决策树(ID3) |
| 统计学习 | 1980s-2000s | SVM、随机森林、AdaBoost |
| 深度学习崛起 | 2010s | AlexNet、GAN、Transformer |
| 大模型时代 | 2020s | GPT、BERT、CLIP、自监督学习 |

### 1.3 基本术语

| 术语 | 含义 |
|------|------|
| 数据集 | 多条记录的集合 |
| 样本 | 数据集中的一条记录 |
| 特征 | 描述样本属性的列 |
| 标签 | 监督学习中的结果 |
| 训练集/验证集/测试集 | 训练模型 / 调超参数 / 评估性能 |
| 参数 | 模型从数据中学到的值（如权重、偏置） |
| 超参数 | 用户手动设定的值（如学习率、K值） |

## 第二章  机器学习基本理论

### 2.1 三要素

$$
\text{机器学习方法} = \text{模型} + \text{策略} + \text{算法}
$$

| 要素 | 含义 | 例子 |
|------|------|------|
| 模型 | 数学描述的参数系统 | 线性模型 \(y=wx+b\) |
| 策略 | 评价模型好坏的标准 | 最小化均方误差 |
| 算法 | 求解最优参数的方法 | 梯度下降、正规方程 |

### 2.2 方法分类

| 分类维度 | 类别 |
|----------|------|
| 按监督信号 | 有监督 / 无监督 / 半监督 / 强化学习 |
| 按模型性质 | 概率模型/非概率模型，线性/非线性 |
| 按学习技巧 | 贝叶斯学习、核方法、集成学习 |

### 2.3 监督学习建模流程

```
原始数据 → 特征工程 → 训练集 → 模型训练 → 模型评估 → 调优 → 最终模型 → 预测
```

### 2.4 特征工程

| 环节 | 方法 | 说明 |
|------|------|------|
| **特征选择** | 过滤法（统计检验）、包裹法（RFE）、嵌入法（L1） | 剔除冗余/无关特征 |
| **特征转换** | 归一化(MinMax)、标准化(Z-score)、对数变换、独热编码 | 使数据适配模型 |
| **特征构造** | 交互特征(乘积)、统计特征(均值/方差)、日期特征(星期/月份) | 创造新特征 |
| **特征降维** | PCA、LDA、t-SNE | 高维→低维，去冗余 |

**归一化 vs 标准化**：

| 方法 | 公式 | 输出 | 适用 |
|------|------|------|------|
| 归一化 | \(x' = \frac{x - \min}{\max - \min}\) | [0,1] | KNN、SVM（对尺度敏感） |
| 标准化 | \(x' = \frac{x - \mu}{\sigma}\) | μ=0, σ=1 | 线性回归、逻辑回归（假设正态分布） |

**独热编码（One-Hot）**：无序类别特征 → 二进制列。设 `drop='first'` 避免多重共线性。

### 2.5 模型评估与选择

#### 2.5.1 损失函数

| 损失函数 | 公式 | 适用 |
|----------|------|------|
| 0-1 损失 | \(\mathbb{I}(y \neq \hat{y})\) | 分类，不可导 |
| 平方损失 | \((y - \hat{y})^2\) | 回归 |
| 绝对损失 | \(\lvert y - \hat{y} \rvert\) | 回归，抗异常值 |
| 对数似然损失 | \(-\log P(y \mid \hat{y})\) | 分类（交叉熵） |

#### 2.5.2 欠拟合与过拟合

| 问题 | 表现 | 原因 | 解决 |
|------|------|------|------|
| **欠拟合** | 训练/测试误差都高 | 模型太简单、特征不足 | 增加特征/复杂度 |
| **过拟合** | 训练误差低、测试误差高 | 模型太复杂、数据少 | 正则化/交叉验证/Dropout/早停 |

#### 2.5.3 正则化

通过惩罚大参数防止过拟合：

| 类型 | 公式 | 效果 | 回归名 |
|------|------|------|--------|
| L1 | \(+\lambda\sum \lvert w_i \rvert\) | 产生**稀疏解**（部分权重→0）= 特征选择 | Lasso |
| L2 | \(+\lambda\sum w_i^2\) | 权重**整体缩小**，不归零 | Ridge（岭回归） |
| ElasticNet | \(+\lambda_1\sum \lvert w_i \rvert + \lambda_2\sum w_i^2\) | 结合两者 | ElasticNet |

#### 2.5.4 交叉验证

| 方法 | 操作 | 特点 |
|------|------|------|
| 简单交叉验证 | 70%训练，30%验证 | 单次划分，不稳定 |
| **k折交叉验证** | 分k份，每次k-1份训练+1份验证，重复k次取平均 | 数据利用率高，结果稳定 |
| 留一交叉验证 | k=样本数（每样本单独验证） | 最充分利用，但计算量大 |

### 2.6 模型求解算法

#### 2.6.1 解析法

直接求解 $\nabla L = 0$，得到闭式解。适用：线性回归的**最小二乘法**（正规方程）。

$$
\theta = (X^T X)^{-1} X^T y
$$

- 优点：一步到位，精确解
- 缺点：$X^TX$ 不可逆或不稳定时失效；$O(n^3)$ 复杂度，特征多时太慢

#### 2.6.2 梯度下降法 ★

$$
\theta \leftarrow \theta - \eta \cdot \nabla_\theta L(\theta)
$$

| 变体 | 每次梯度用多少数据 | 特点 |
|------|--------------------|------|
| BGD（批量） | 全部数据 | 稳定，但慢 |
| SGD（随机） | 1条数据 | 快但震荡 |
| **MBGD（小批量）** | 若干条数据 | **折中最常用** |

#### 2.6.3 牛顿法（了解）

利用二阶导数（Hessian矩阵）加速收敛：$\theta \leftarrow \theta - H^{-1}\nabla L$。收敛快但计算Hessian成本高。

### 2.7 模型评价指标

#### 回归指标

| 指标 | 公式 | 特点 |
|------|------|------|
| MAE | \(\frac{1}{n}\sum \lvert y-\hat{y} \rvert\) | 抗异常值，量纲一致 |
| MSE | \(\frac{1}{n}\sum(y-\hat{y})^2\) | 惩罚大误差 |
| RMSE | \(\sqrt{MSE}\) | 量纲与目标一致 |
| \(R^2\) | \(1 - \frac{SS_{res}}{SS_{tot}}\) | 解释能力，越接近1越好 |

#### 分类指标

**混淆矩阵**（二分类）：

| 真实 \ 预测 | 正例 | 负例 |
|-------------|------|------|
| 正例 | TP | FN |
| 负例 | FP | TN |

| 指标 | 公式 | 含义 |
|------|------|------|
| 准确率 | \(\frac{TP+TN}{TP+TN+FP+FN}\) | 整体正确率（类别不均衡时失效） |
| 精确率 | \(\frac{TP}{TP+FP}\) | 预测为正的有多少真为正 |
| 召回率 | \(\frac{TP}{TP+FN}\) | 真正的正例被找出多少 |
| F1 | \(2\cdot\frac{P\cdot R}{P+R}\) | 精确率与召回率的调和平均 |

**ROC曲线与AUC**：横轴=假正率(FPR)，纵轴=真正率(TPR)。AUC越大模型越好（AUC=1 完美，AUC=0.5 随机猜测）。

## 第三章  KNN算法

### 3.1 核心思想

找距离最近的 K 个邻居，投票（分类）或平均（回归）决定结果。

$$
\text{分类：} \quad \hat{y} = \text{mode}(\{y_i | i \in \text{Neighbors}_K(x)\})
$$

$$
\text{回归：} \quad \hat{y} = \frac{1}{K}\sum_{i \in \text{Neighbors}_K(x)} y_i
$$

### 3.2 距离度量

| 距离 | 公式 | 特点 |
|------|------|------|
| 欧氏距离 | \(\sqrt{\sum(x_i - y_i)^2}\) | 直线距离，最常用 |
| 曼哈顿距离 | \(\sum \lvert x_i - y_i \rvert\) | 网格路径 |
| 切比雪夫距离 | \(\max \lvert x_i - y_i \rvert\) | 国际象棋国王步数 |
| 闵可夫斯基距离 | \((\sum \lvert x_i-y_i \rvert ^p)^{1/p}\) | p=2→欧氏, p=1→曼哈顿 |

### 3.3 优缺点

| 优点 | 缺点 |
|------|------|
| 简单直观，无需训练 | 计算量大（每次预测需遍历全集） |
| 无参数学习（非参数模型） | 对尺度敏感 → 必须归一化 |
| 可用于分类和回归 | K 值选择敏感 |

### 3.4 案例：心脏病预测

**流程**：加载数据 → 特征工程（标准化+独热编码+去共线性）→ `KNeighborsClassifier(n_neighbors=3)` → 评估

**超参数调优**：`GridSearchCV` 遍历 `n_neighbors=[1..9]` + 10折交叉验证。

## 第四章  线性回归

### 4.1 模型

**一元**：$y = wx + b$

**多元**：$y = \theta_1 x_1 + \theta_2 x_2 + ... + \theta_n x_n + b = \theta^T x + b$

### 4.2 损失函数：均方误差（MSE）

$$
J(\theta) = \frac{1}{n}\sum_{i=1}^n (y_i - \hat{y}_i)^2
$$

基于 MSE 最小化求解的方法称**最小二乘法**。数学原理：误差服从正态分布时，MSE 最小化等价于极大似然估计。

### 4.3 两种求解方法

| 方法 | 正规方程法（解析法） | 梯度下降法 |
|------|---------------------|-----------|
| 原理 | \(\theta = (X^TX)^{-1}X^Ty\) | \(\theta \leftarrow \theta - \eta\nabla J\) |
| 优点 | 一步求解，精确 | 可处理大数据/高维 |
| 缺点 | \(O(n^3)\)，特征多时慢；矩阵不可逆时失效 | 需调学习率，可能局部最优 |
| API | `LinearRegression()` | `SGDRegressor()` |

### 4.4 特征缩放的重要性

特征尺度不一致 → 梯度下降路径扭曲、收敛慢 → 需先标准化/归一化。

### 4.5 案例：广告投放效果预测

- **任务**：根据 TV/Radio/Newspaper 广告预算 → 预测销售额
- **方法**：标准化 → `LinearRegression()` 或 `SGDRegressor()` → MSE 评估

## 第五章  逻辑回归

### 5.1 模型

虽是"回归"之名，实为**分类**。将线性输出映射为概率：

$$
P(y=1|x) = \sigma(\theta^T x + b) = \frac{1}{1 + e^{-(\theta^T x + b)}}
$$

决策：$P > 0.5$ → 1类，否则 0类。

### 5.2 损失函数：对数损失（BCE）

$$
J(\theta) = -\frac{1}{n}\sum_{i=1}^n [y_i\log\hat{y}_i + (1-y_i)\log(1-\hat{y}_i)]
$$

- 来源：极大似然估计 → 取负对数 → 最小化
- 梯度：$\nabla J = \frac{1}{n}X^T(\hat{y} - y)$（形式与线性回归一致！）

### 5.3 多分类扩展

| 方法 | 原理 | API |
|------|------|-----|
| **一对多（OvR）** | 训练 C 个二分类器（每类 vs 其他） | `multi_class='ovr'` |
| **Softmax回归** | 一个模型输出 C 类概率（\(\frac{e^{z_i}}{\sum e^{z_k}}\)） | `multi_class='multinomial'`（默认） |

### 5.4 案例：手写数字识别

28×28 像素 → 784 维特征 → Softmax 逻辑回归 → 10 类输出。

## 第六章  感知机

### 6.1 模型

$$
y = \begin{cases} 1 & (w_1 x_1 + w_2 x_2 + b > 0) \\ 0 & (\text{otherwise}) \end{cases}
$$

### 6.2 逻辑电路

| 门 | \(x_1\) | \(x_2\) | 输出 | 线性可分？ |
|----|-------|-------|------|-----------|
| AND | 0/1 | 0/1 | 全1→1 | ✓ |
| NAND | 0/1 | 0/1 | 全1→0 | ✓ |
| OR | 0/1 | 0/1 | 有1→1 | ✓ |
| XOR | 0/1 | 0/1 | 不同→1 | ✗ |

### 6.3 局限性

单层感知机**只能分离线性可分数据**（XOR 不可分）。

→ 解决方案：**多层感知机（MLP）**，加入隐藏层 + 非线性激活函数。

## 第七章  其他监督学习算法（了解）

### 7.1 朴素贝叶斯

基于贝叶斯定理 + "特征条件独立"假设：

$$
P(C|X) \propto P(C) \cdot \prod_{i} P(x_i | C)
$$

- 简单高效，适合文本分类（垃圾邮件过滤）
- 假设过强，现实中特征往往不独立

### 7.2 决策树

树形结构，每个内部节点对一个特征做判断，分支=判断结果，叶节点=分类/回归结果。

| 算法 | 划分标准 | 特点 |
|------|----------|------|
| ID3 | 信息增益 | 偏向取值多的特征 |
| C4.5 | 信息增益率 | 修正 ID3 偏差 |
| CART | 基尼指数(分类)、MSE(回归) | 二叉树，可分类可回归 |

- 可解释性强，无需特征缩放
- 容易过拟合 → 需要**剪枝**（预剪枝/后剪枝）

### 7.3 支持向量机（SVM）

找**最大间隔**超平面分离类别：

| 情况 | 方法 |
|------|------|
| 线性可分 | 硬间隔最大化 |
| 近似线性可分 | 软间隔（允许少量误分类，加松弛变量） |
| 非线性 | **核函数**（RBF、多项式）映射到高维空间 |

### 7.4 集成学习

组合多个弱学习器 → 强学习器。

| 方法 | 策略 | 代表 |
|------|------|------|
| **Boosting** | 串行，后一轮关注前一轮的错分样本 | AdaBoost |
| **Bagging** | 并行，自助采样+投票 | **随机森林** |

**随机森林**：多棵决策树 + 随机抽样（样本+特征）→ 投票。抗过拟合、可输出特征重要性。

## 第八章  无监督学习

### 8.1 聚类

无标签，根据相似性将数据自动分组。

#### K-means（K均值聚类）

$$
\min \sum_{k=1}^K \sum_{x_i \in C_k} \|x_i - \mu_k\|^2
$$

**流程**：随机选K个中心 → 分配样本到最近中心 → 重新计算中心 → 迭代至收敛

| 优点 | 缺点 |
|------|------|
| 简单快速 | 需预设 K 值 |
| 可扩展 | 对初始中心敏感 |
| | 球形簇假设（非球形效果差） |

**K值选择**：肘部法（观察簇内平方和曲线拐点）、轮廓系数。

#### 层次聚类

- 自底向上（凝聚）：每个样本开始→逐步合并
- 自顶向下（分裂）：全部样本开始→逐步分裂

#### DBSCAN（密度聚类）

无需预设 K，基于密度可达：核心对象 + $\epsilon$-邻域 + MinPts。能发现任意形状的簇，自动识别噪声。

### 8.2 降维

高维 → 低维，保留主要信息。

#### PCA（主成分分析）

找方差最大的投影方向（主成分），将数据投影到低维空间。

- 步骤：中心化 → 协方差矩阵 → 特征值分解 → 取前k个最大特征值对应的特征向量
- 也可用 SVD（奇异值分解）实现

#### t-SNE

非线性降维，适合**可视化**（将高维数据映射到2D/3D）。保留局部结构但不保留全局距离。
