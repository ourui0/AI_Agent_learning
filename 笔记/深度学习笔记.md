## 第一章  深度学习概述

### 1.1 概念关系

```
AI → ML → DL（多层神经网络）
```

- **深度学习**是机器学习的子集，使用多层神经网络自动提取层次化特征
- 人脑神经元启发：树突（加权输入）→ 细胞体（求和+偏置）→ 轴突（激活函数输出）
- "深度"指隐藏层数量：浅层学边缘/纹理 → 深层学语义/部件

### 1.2 特点对比

| 传统 ML | 深度学习 |
|---------|----------|
| 人工特征工程 | 自动特征提取 |
| 小样本即可 | 大数据效果显著 |
| CPU 可行 | 需 GPU |
| 可解释性好 | 黑箱 |

### 1.3 三大驱动力

- **算法**：反向传播(1986) → ReLU(2011) → BN/Dropout(2014) → ResNet(2015) → Transformer(2017)
- **数据**：ImageNet、互联网文本、传感器数据
- **算力**：NVIDIA CUDA GPU、TPU、分布式训练

### 1.4 主要应用

计算机视觉、NLP（翻译/对话/生成）、推荐系统、医疗影像、自动驾驶、金融风控

## 第二章  神经网络基础

### 2.1 神经元模型

$$
a = \sum_{i=1}^{n} w_i x_i + b, \quad y = \sigma(a)
$$

- $x_i$：输入，$w_i$：权重（可学习），$b$：偏置，$\sigma$：激活函数

### 2.2 网络结构

```
输入层 → 隐藏层(≥1) → 输出层
```

- 全连接（FC）：下层每神经元连接上层**所有**神经元
- 前向传播：逐层计算，上一层的输出 = 下一层的输入
- 权重矩阵形状：若第 $l$ 层 $m$ 个神经元、第 $l+1$ 层 $n$ 个，则 $W^{(l)}$ 为 $m \times n$

### 2.3 感知机 → 激活函数

感知机：

$$
y = \begin{cases}
1 & (\sum w_i x_i + b > 0) \\
0 & (\text{otherwise})
\end{cases}
$$

问题：线性不可分（XOR）。引入非线性激活函数 → 可拟合任意函数。

### 2.4 激活函数详解

| 函数 | 公式 | 输出 | 特点 |
|------|------|------|------|
| Sigmoid | \(\dfrac{1}{1+e^{-x}}\) | (0,1) | 平滑、梯度消失（导数≤0.25）、非零中心 |
| Tanh | \(\dfrac{e^x-e^{-x}}{e^x+e^{-x}}\) | (-1,1) | 零中心、仍有梯度消失 |
| **ReLU** | \(\max(0,x)\) | \([0,+\infty)\) | 计算快、正区间梯度=1（**解决梯度消失**）、Dead Neuron |
| LeakyReLU | \(\max(\alpha x, x)\) | \((-\infty,+\infty)\) | 修复 Dead Neuron |
| **Softmax** | \(\dfrac{e^{x_i}}{\sum e^{x_k}}\) | (0,1), Σ=1 | 概率分布，多分类输出层专用 |

**选型速记**：隐藏层默认 ReLU（有问题换 LeakyReLU）；二分类输出用 Sigmoid；多分类输出用 Softmax。

### 2.5 输出层设计

| 任务 | 激活 | 神经元数 |
|------|------|----------|
| 二分类 | Sigmoid | 1 |
| 多分类 | Softmax | 类别数 |
| 回归 | 恒等函数 | 目标维度 |

## 第三章  神经网络的学习

### 3.1 损失函数

衡量预测 $\hat{y}$ 与真实 $y$ 的差距，训练目标 = 最小化损失。

| 损失函数 | 公式 | 适用 |
|----------|------|------|
| MSE (L2) | \(\frac{1}{n}\sum(y_i-\hat{y}_i)^2\) | 回归，对异常值敏感 |
| MAE (L1) | \(\frac{1}{n}\sum \lvert y_i-\hat{y}_i \rvert\) | 回归，抗异常值 |
| BCE | \(-[y\log\hat{y}+(1-y)\log(1-\hat{y})]\) | 二分类 + Sigmoid |
| CrossEntropy | \(-\sum t_k \log \hat{y}_k\) | 多分类 + Softmax |

**Softmax+CrossEntropy** 是多分类标配，反向传播时梯度为简洁的 $\hat{y} - t$。

### 3.2 数值微分与梯度

**中心差分**：$f'(x) \approx \dfrac{f(x+h) - f(x-h)}{2h}, \; h = 10^{-4}$

**梯度**：所有偏导数组成的向量 $\nabla f = (\frac{\partial f}{\partial x_1}, ..., \frac{\partial f}{\partial x_n})$

- 梯度指向函数**增大最快**的方向
- 负梯度指向函数**减小最快**的方向（训练走这个方向）
- 极小/极大/鞍点处梯度为 0

### 3.3 梯度下降（SGD）

$$
\theta \leftarrow \theta - \eta \cdot \nabla_\theta L(\theta)
$$

- $\theta$：参数，$\eta$：学习率（超参数）
- $\eta$ 太大 → 震荡发散；太小 → 收敛慢

**核心概念**：

| 术语 | 含义 |
|------|------|
| Epoch | 完整遍历一次训练集 |
| Batch Size | 每次更新用的样本数 |
| Iteration | 完成一次前向+反向+更新 |

### 3.4 训练循环

```
for epoch:
    for batch in data:
        ŷ = forward(x)            # 前向
        loss = loss_fn(ŷ, y)      # 损失
        grad = backward(loss)     # 反向传播（求梯度）
        params -= lr * grad       # 更新
```

## 第四章  反向传播

### 4.1 计算图与链式法则

将计算表示为 DAG（有向无环图），节点=运算，边=数据流。

**链式法则**：$\dfrac{\partial L}{\partial x} = \dfrac{\partial L}{\partial y} \cdot \dfrac{\partial y}{\partial x}$

从输出端逐层将梯度反向传播至输入端，一次反向传播算完所有参数的梯度（vs 数值微分需逐个参数算）。

### 4.2 关键节点的反向传播

| 节点 | 前向 | 反向（局部梯度） |
|------|------|------------------|
| 加法 | \(z = x + y\) | 梯度原样传递（×1） |
| 乘法 | \(z = x \cdot y\) | 梯度互换传递（×对方值） |
| ReLU | \(y = \max(0, x)\) | \(x>0\) 时传梯度，\(x\leq0\) 时截断 |
| Sigmoid | \(y = \sigma(x)\) | \(y(1-y)\) × 上游梯度 |
| Softmax+CE | 联合计算 | \(\hat{y} - t\)（极简洁） |
| Affine | \(y = xW + b\) | \(\frac{\partial L}{\partial x} = \frac{\partial L}{\partial y} \cdot W^T\) |

## 第五章  训练技巧

### 5.1 梯度消失与梯度爆炸

- **梯度消失**：深层网络梯度过小，浅层参数几乎不更新 → 用 ReLU + BN + 残差连接
- **梯度爆炸**：梯度过大，参数剧烈震荡 → **梯度裁剪**（`clip_grad_norm_`）

### 5.2 优化器演进

| 优化器 | 核心改进 | 说明 |
|--------|----------|------|
| SGD | \(w \leftarrow w - \eta \nabla w\) | 基准，收敛慢 |
| Momentum | \(v \leftarrow \beta v - \eta \nabla w\) | 积累梯度惯性，减少震荡 |
| AdaGrad | \(h \leftarrow h + (\nabla w)^2\) | 自适应学习率，稀疏特征好，后期衰减过度 |
| RMSProp | \(h \leftarrow \rho h + (1-\rho)(\nabla w)^2\) | 指数滑动平均替代累加 |
| **Adam** | 结合 Momentum + RMSProp | **当前最常用**，自适应 + 惯性 |

### 5.3 学习率调度

- 等间隔衰减、指数衰减、余弦退火（Cosine Annealing）
- 目的：前期大步快走，后期小步精调

### 5.4 参数初始化

| 方法 | 适用 | 说明 |
|------|------|------|
| Xavier | Sigmoid/Tanh | \(W \sim U(-\sqrt{6/(n_{in}+n_{out})}, ...)\) |
| **He (Kaiming)** | **ReLU** | \(W \sim N(0, \sqrt{2/n_{in}})\) |

### 5.5 正则化

| 方法 | 原理 |
|------|------|
| **Dropout** | 训练时随机失活神经元（类似集成学习），防止共适应 |
| **Batch Normalization** | 每层输入标准化，加速训练，允许更高学习率 |
| 权值衰减 (L2) | 损失 + \(\lambda\|W\|^2\)，限制权重规模 |

```python
x.grad                   # 获取梯度
```

### 6.3 构建模型

- 继承 `nn.Module`：重写 `__init__` 和 `forward()`
- `nn.Sequential`：顺序堆叠层
- 关键层：`nn.Linear`, `nn.Embedding`, `nn.Conv2d`, `nn.RNN`, `nn.Dropout`, `nn.BatchNorm1d`

### 6.4 训练模板（PyTorch）

```python
model = MyModel()
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=1e-3)

for epoch in range(epochs):
    for x, y in dataloader:
        optimizer.zero_grad()
        loss = criterion(model(x), y)
        loss.backward()
        optimizer.step()
```

## 第七章  卷积神经网络（CNN）

### 7.1 核心思想

**局部连接 + 权值共享**：一个卷积核扫描全图，大幅减少参数量。

全连接层的问题是**忽视空间信息**：28×28 的图像被拉平成 784 维向量，相邻像素的位置关系丢失。CNN 保持 3D 形状（$C \times H \times W$）输入和输出，卷积核只在局部感受野内操作。

### 7.2 卷积层

输入 $(C_{in}, H, W)$ → 卷积核 $(C_{out}, C_{in}, kH, kW)$ → 输出 $(C_{out}, H', W')$

$$
H' = \left\lfloor\frac{H + 2P - K}{S}\right\rfloor + 1
$$

| 参数 | 含义 | 作用 |
|------|------|------|
| \(K\) (kernel_size) | 卷积核尺寸 | 控制感受野大小 |
| \(P\) (padding) | 边缘填充（通常填 0） | 控制输出尺寸，保持空间信息 |
| \(S\) (stride) | 滑动步幅 | \(S>1\) 时下采样 |
| \(C_{out}\) | 输出通道数（= 卷积核个数） | 每个核提取一种特征 |

- **权值共享**：同一卷积核的参数在所有位置复用——参数量与输入尺寸无关
- 偏置：每个输出通道一个标量偏置 $b$

**API**：`nn.Conv2d(in_channels, out_channels, kernel_size, stride, padding)`

### 7.3 池化层

降采样，缩小 $H \times W$：

| 类型 | 操作 | 特点 |
|------|------|------|
| Max Pooling | 取窗口内最大值 | 保留显著特征，常用 |
| Avg Pooling | 取窗口内平均值 | 平滑，保留背景信息 |

**API**：`nn.MaxPool2d(kernel_size, stride)` / `nn.AvgPool2d(kernel_size, stride)`

作用：减少计算量、扩大感受野、增强平移不变性。

### 7.4 经典 CNN 架构演进

| 网络 | 年份 | 层数 | 核心创新 |
|------|------|------|----------|
| LeNet | 1998 | 5 | 卷积+池化+全连接，开山之作 |
| **AlexNet** | 2012 | 8 | ReLU + Dropout + GPU，**深度学习复兴标志** |
| VGG | 2014 | 16/19 | 小卷积核(3×3) 堆叠，结构简洁 |
| GoogleNet | 2014 | 22 | Inception 结构，横向多尺度并行 |
| **ResNet** | 2015 | 50/101/152 | **残差连接**：\(H(x) = F(x) + x\)，解决深层梯度消失 |

ResNet 的残差连接（跳跃连接）是最重要的创新：网络学习目标从 $H(x)$ 变为残差 $F(x) = H(x) - x$，恒等映射 $x$ 直接跨层传递，梯度不会消失。

### 7.5 案例：服装分类（Fashion-MNIST）

> 配套代码：`CNN/4_fashion.py`

**数据集**：28×28 灰度图，10 类（T恤/裤子/套头衫/连衣裙/外套/凉鞋/衬衫/运动鞋/包/靴子）

**模型结构（类 LeNet）**：

```
Input (1, 28, 28)
   ↓
Conv2d(1→6, k=5, p=2)   → (6, 28, 28)     # padding=2 保持尺寸
Sigmoid
AvgPool2d(k=2, s=2)     → (6, 14, 14)     # 尺寸减半
   ↓
Conv2d(6→16, k=5)       → (16, 10, 10)    # 无padding，尺寸缩减
Sigmoid
AvgPool2d(k=2, s=2)     → (16, 5, 5)      # 尺寸再减半
   ↓
Flatten                 → (400,)           # 16×5×5 = 400
Linear(400→120) + Sigmoid
Linear(120→84) + Sigmoid
Linear(84→10)                              # 10 类输出
```

**形状推导**（以单张 28×28 灰度图为例）：

| 层 | 输入形状 | 输出形状 | 计算公式 |
|----|---------|----------|----------|
| Conv2d(1,6,5,1,2) | (1,28,28) | (6,28,28) | \((28+4-5)/1+1=28\) |
| AvgPool(2,2) | (6,28,28) | (6,14,14) | \(28/2=14\) |
| Conv2d(6,16,5) | (6,14,14) | (16,10,10) | \((14+0-5)/1+1=10\) |
| AvgPool(2,2) | (16,10,10) | (16,5,5) | \(10/2=5\) |
| Flatten | (16,5,5) | (400,) | \(16\times5\times5=400\) |
| Linear(400,120) | (400,) | (120,) | |
| Linear(120,84) | (120,) | (84,) | |
| Linear(84,10) | (84,) | (10,) | |

**训练配置**：
- 损失函数：`CrossEntropyLoss`（多分类）
- 优化器：`SGD(lr=0.9)`
- 参数初始化：`Xavier Uniform`
- 批量大小：256
- 训练轮次：20 epoch

**训练流程**（每个 epoch）：
1. `model.train()` — 训练模式（启用 Dropout 等）→ 遍历训练集、反向传播、更新参数
2. `model.eval()` + `torch.no_grad()` — 评估模式 → 在测试集上计算准确率
3. 输出：`loss + train_acc + test_acc`

> ⚠️ 此案例用 Sigmoid + AvgPool + SGD，是经典 LeNet 风格。实际工程中通常替换为 ReLU + MaxPool + Adam，效果更好。

## 第八章  循环神经网络（RNN）

### 8.1 文本表示

- 同义词词典 → 计数/共现矩阵 → 分布式表示（词向量）
- **词嵌入（Embedding）**：离散 ID → 稠密向量，语义相近的词向量距离近

### 8.2 RNN 核心公式

$$
h_t = \tanh(W_h h_{t-1} + W_x x_t + b)
$$

$$
y_t = W_y h_t + b_y
$$

- 隐藏状态 $h_t$ 在时间步之间传递 → "记忆"
- 问题：梯度消失/爆炸、长序列建模差

### 8.3 RNN 应用模式

| 模式 | 输入→输出 | 例子 |
|------|-----------|------|
| one-to-one | 固定 → 固定 | 图像分类 |
| one-to-many | 固定 → 序列 | 图像描述 |
| many-to-one | 序列 → 固定 | 情感分类 |
| many-to-many | 序列 → 序列 | 机器翻译、文本生成 |

### 8.4 古诗生成案例

```
Embedding(vocab_size, 256) → RNN(256, 512, 2层) → Linear(512, vocab_size)
                                                      ↓
                                               CrossEntropyLoss
```

训练后用起始字逐字采样生成诗句。
